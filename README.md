# LeetCode Patterns for Coding Interviews

This repository contains my solutions to **LeetCode problems organized by problem-solving patterns**.  
Instead of solving problems randomly, I focus on **learning common algorithmic patterns** that frequently appear in coding interviews.

The goal is to improve problem-solving skills in **Data Structures & Algorithms (DSA)** and prepare for **technical interviews**.

---

## Why Patterns?

Most coding interview questions follow common patterns.  
Once you recognize the pattern, solving problems becomes much easier.
You know which approach to use when or when not

This repository organizes problems into categories such as:

- Two Pointers
- Sliding Window
- Binary Search
- Prefix Sum
- Heap
- Recursion & Backtracking
- Tree Patterns
